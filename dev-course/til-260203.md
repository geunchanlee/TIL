## String 클래스

기본 char 타입은 문자 한 개 저장에 쓰임.

여러 개의 문자열을 저장하기 위해 String 클래스 제공

string 클래스는 new 키워드 없이 참조 타입으로 기본 타입처럼 대입연산자 사용함

## 형변환 type conversion

형 변환의 기본 원칙

1. 바이트 크기가 작은 자료형에서 큰 자료형으로 형 변환은 자동으로 이루어진다.
2. 덜 정밀한 자료형에서 더 정밀한 자료형으로 형 변환은 자동으로 이루어진다.

`(type)` 을 사용해 명시적 형 번환 가능

```java
	double dNum1 = 2.2;
	float fNum2 = 0.9f;
	
	int iNum3 = (int)dNum1 + (int)fNum2;
	int iNum4 = (int)(dNum1 + fNum2);
	System.out.println(iNum3);
	System.out.println(iNum4);

	>>> 2
	>>> 3
```

## 연산자

### **산술 연산자**

`+`, `-`, `*`, `/`, `%`

**나누기 연산의 오류**

`24 / 7` 같은 연산을 하면 의도와 다른 값이 나올 수 있다.

모든 변수(계산되는 변수, 계산 결과를 담는 변수)의 타입이 부동소수점일 때 정확한 나누기 값을 알 수 있다.

```java
	System.out.println(24 / 7);
    >>> 3

	System.out.println(24.0 / 7.0);
	>>> 3.4285714285714284
```

### 복합 대입 연산자

## 조건문

### if

주어진 조건에 따라 다른 문장 선택하는 제어문

```java 
if ( boolean or condition ) { 
    statement; 
}
``` 

condition은 비교문, statement: stmt는 실행문이다.

### if ~ else if ~ else

```java
if ( condition1 ) {
    stmt;
} else if ( condition2 ) {
    stmt;
} else {
    stmt;
}

```


> 조건문을 작성할 땐 코드를 먼저 작성하지 말고 분기 조건을 손으로 정리한 다음에 작성하자.
> 

코드 구현 전엔 항상 먼저 정리한 다음에 코드로 옮기는 습관을 들이자.

### switch ~ case

조건식의 결과가 정수 또는 문자열 값이고 그 값에 따라 수행결과가 다른 경우 사용

```java
switch ( expr ) { 
	case value1 :
        stmt;
		break;
	case value2 :
        stmt;
		break;
	case value3 :
        stmt;
		break;
    ...
	default: 
        stmt;
        break;
}
```

## 반복문

반복할 때 사용

배열처럼 연속 데이터를 조회할 때 주로 사용

### while

```java
while( condition ) { 
    stmt; 
}
```

condition 이 참일 때 반복한다.

### do ~ while

```java
do {
    stmt;
} while ( condition ); // 세미콜론 필수
```

do 안의 문장을 무조건 한 번은 수행한 후 반복 조건을 검사한다.

do-while은 반복문 중 유일하게 맨 뒤에 세미콜론을 꼭 넣어야 하는 것 조심하기.

### for

for문은 초기화, 조건비교, 증감식을 한 줄로 쓸 수 있어 많이 사용

```java
for( init; condition; increment ) {
    stmt;
}
```
for 반복문 안에서 초기화, 조건 비교, 증감이 한 눈에 보인다.
