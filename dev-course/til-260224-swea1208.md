# TIL 26/02/24 - SWEA 1208. Flatten

문제 링크: [Flatten](https://swexpertacademy.com/main/code/problem/problemDetail.do?contestProbId=AV139KOaABgCFAYh) / D3

**문제**

- 100개의 상자 높이가 숫자로 주어진다. 높은 곳의 상자를 낮은 곳에 옮기는 작업을 덤프라고 정의한다.
- 상자의 최고 높이와 최저 높이 차가 최대 1 이내면 평탄화가 완료된다.
- 10개의 테스트 케이스에서 첫 줄에 덤프 횟수가 주어진다.
- 테스트 케이스 두 번째 줄에는 상자들의 높이가 주어진다.
- 주어진 횟수의 덤프를 시행한 후 상자의 최고 높이와 최저 높이의 차를 출력한다.
- 주어진 덤프 횟수 이내에 평탄화가 완료되었을 경우 바로 최고 높이와 최저 높이의 차를 반환한다.

**해결 방법**

1. 우선 상자를 입력받아 배열에 저장한다.
2. 매 반복마다 현재 시점 상자의 최고 높이와 최저 높이를 구한 다음 그 차이가 1 이하라면 평탄화가 완료된 것이므로 높이 차이를 반환하고 종료한다.
3. 그렇지 않다면 덤프 작업을 진행한다. 최고 높이에서 1을 빼고, 최저 높이에 1을 더해 상자의 높이 차를 줄여준다.
4. 주어진 덤프 횟수만큼 반복하고 결과값을 반환한다.

> **input**
>
>	834
>	42 68 35 1 70 25 79 59 63 65 6 46 82 28 62 92 96 43 28 37 92 5 3 54 93 83 22 17 19 96 48 27 72 39 70 13 68 100 36 95 4 12 23 34 74 65 42 12 54 69 48 45 63 58 38 60 24 42 30 79 17 36 91 43 89 7 41 43 65 49 47 6 91 30 71 51 7 2 94 49 30 24 85 55 57 41 67 77 32 9 45 40 27 24 38 39 19 83 30 42

### Python 코드

```python
for t in range(10):
	dump = int(input())
	boxes = [int(i) for i in input().split()] # 상자 입력받기 
	for i in range(dump):
		# 최고, 최저 높이 구해두기
		max_v = max(boxes)
		min_v = min(boxes)
		
		# 평탄화가 이미 완료되었는지 검사
		if max_v - min_v <= 1:
			print(f'#{t} {max_v - min_v}')
			break
		# 덤프 작업 진행
		max_idx = boxes.index(max_v)
		min_idx = boxes.index(min_v)
		boxes[max_idx] -= 1
		boxes[min_idx] += 1
	# break로 반복문이 끝나지 않았을 때 출력
	else:
		print(f'#{t} {max(boxes) - min(boxes)}')

>>> #1 13
```

`max(boxes)`, `min(boxes)`로 상자의 최고, 최저 높이를 구한다.
if문에서 최고높이 - 최저높이가 1 이하라면 평탄화가 완료된 것이기 때문에 결과를 출력하고 반복을 종료한다.
그렇지 않다면 덤프 작업을 진행한다. 최고높이, 최저높이의 인덱스를 구한 뒤 해당 인덱스의 값을 1씩 더하고 줄인다. 

`for - else`로 break로 반복문이 종료되지 않았을 때(반복이 끝까지 수행되었을 때) -> 주어진 덤프 횟수를 모두 사용해 평탄화를 진행한 경우의 결과를 출력해준다.

### Java 코드

```java
import java.util.Scanner;  
  
class Solution {  
    public static void main(String args[]) throws Exception {  
        Scanner sc = new Scanner(System.in);  
        
        for (int test_case = 1; test_case <= 10; test_case++) {  
            int dump = sc.nextInt();  
            
            // 박스높이 배열 입력받기
            int[] boxes = new int[100];  
            for (int i = 0; i < 100; i++) {  
                boxes[i] = sc.nextInt();  
            }  
            
            for (int d = 0; d < dump; d++) {  
				// 최대높이, 최소높이 인덱스 구하기
			    int max_idx = 0;  
                int min_idx = 0;  
                for (int i = 1; i < 100; i++) {  
                    if(boxes[i] > boxes[max_idx]) max_idx = i;  
                    if(boxes[i] < boxes[min_idx]) min_idx = i;  
                }  
                // 이미 평탄화 완료된경우 반복종료
                if (boxes[max_idx] - boxes[min_idx] <= 1) {  
                    break;  
                }
                
                // 덤프 실행
                boxes[max_idx]--;  
                boxes[min_idx]++;  
            }  
            
            // 덤프 다 끝난 후 다시 최대, 최소 인덱스 찾아서 높이 차 출력하기
            int max_idx = 0;  
            int min_idx = 0;  
            for (int i = 1; i < 100; i++) {  
                if (boxes[i] > boxes[max_idx]) max_idx = i;  
                if (boxes[i] < boxes[min_idx]) min_idx = i;  
            }  
            System.out.println("#" + test_case + " " + (boxes[max_idx] - boxes[min_idx]));  
        }  
    }  
}
```

> [!tldr] 출력 결과
```text
#1 13
```

우선 박스 배열을 입력받고 덤프 횟수로 반복문을 돌린다.
최대높이와 최소높이를 반복문을 통해 박스 배열을 순회하면서 구한다. 
-> 인덱스를 구했기 때문에 값에도 접근할 수 있다.
> 처음에는 for-each로 최대, 최소값을 구한 다음 그 값인 인덱스를 다시 또 구하는 코드를 작성했다. 코드를 작성하기 전에 필요한 반복문 조건을 먼저 종이에 고민해보는게 중요할 것 같다.

최대, 최소위치를 구한 다음 파이썬 코드처럼 평탄화가 이미 되었는지 검사하고, 아니라면 덤프를 실행한다.
정해진 횟수만큼 덤프를 실행하는동안 반복하고 모든 덤프가 종료되면 최대, 최소값의 차를 출력한다.
