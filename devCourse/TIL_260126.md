# TIL 26/01/26 - 전역 CSS 변수 사용하기

프로젝트 진행 중 현직 멘토님께 멘토링 받을 때 팀원분이 질문을 했다.

팀에서 공동으로 사용하는 스타일 등을 정해두는 범위가 어떻게 되는지?

프로젝트마다, 회사마다 다르지만 개발하면서 스타일을 맞춰나가는 기준은 많은 팀이 가지고 있다고 한다.

우리도 프로젝트 진행하면서 처음에 서로 아무렇게 만들었더니 모양도, 색도 제각각으로 나와서 모두가 공유할 `style.css` 를 하나 만들고 가장 기초적인 내용을 담아두기로 했다.

<br/>

## style.css

```css
@import url('https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/static/pretendard.css');
@import url('https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,100..900;1,100..900&display=swap');

:root {
    --bg-color: #fffffa; /* 흰색 배경 */
    --beige-bg: #F4EFEB; /* 박스 배경 */
    --primary-text: #333333; /* 글씨 색상*/
    --secondary-text: #666666; /* 글씨 색상 */
    --alt-white: #fafafa; /* #ffffff 대체 흰색 */
    --alt-black: #121212; /* #000000 대체 검은색 */
    --navy-blue: #2f4157; /* 버튼 색상 등등 */
    --btn-red: #b93920; /* 로그인 버튼, 삭제 버튼 등 색상 */
    --border-radius-lg: 30px; /* 모서리 둥글기 값 */
}

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Pretendard', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
    background-color: var(--bg-color);
    color: var(--primary-text);
}
```

팀원 모두가 공유할 내용들이 담긴 파일이다.

첫 부분에서 사용할 폰트를 불러오고 있고, *에서 페이지 구성을 위한 기본적인 설정을 진행한다.

`:root`가 오늘 배운 내용의 핵심인데 주로 우리가 사용해야 할 몇 가지 컬러 값과 모서리 둥글기를 변수로 선언해서 지정해두고 아래 body 부분에서 사용하고 있다.

각자 작업하는 html파일의 첫머리에 우선 이 `style.css`를 불러오고, 본인이 작업할 페이지의 스타일을 사용하면 모두가 통일감있는 스타일을 적용할 수 있게 된다.

<br/>

## :root

[MDN Web Docs - :root](https://developer.mozilla.org/ko/docs/Web/CSS/Reference/Selectors/:root)

`:` 는 가상 클래스이다. 선택자에 추가되는 요소인데 선택한 요소가 특정한 상태일 때 만족하는 키워드이다. 지금까진 주로 버튼 등에서 마우스를 hover했을때 속성을 `btn:hover` 처럼 정해줄 때 많이 사용했었다.

`:root`는 html문서의 `<html>` 선택자와 똑같다고 한다. 하지만 `:root`로 루트 요소를 선택하면 `html`보다 더 높은 명시도를 갖는다. → 여기에 속성을 정의하면 모든 요소에서 변수를 사용할 수 있다.

공식문서의 예제에서도 바로 전역 변수 선언에 유용하게 사용할 수 있다고 보여주고 있다.

<br/>

## 사용자 지정 css 속성

[MDN Web Docs - css 변수](https://developer.mozilla.org/ko/docs/Web/CSS/Guides/Cascading_variables/Using_custom_properties)

`--` 를 이용해서 사용자 지정 CSS 속성을 사용할 수 있다.

CSS에서 변수를 정의해서 사용한다고 생각하면 될 것 같다.

```css
:root {
    --bg-color: #fffffa; /* 흰색 배경 */
    --beige-bg: #F4EFEB; /* 박스 배경 */
    ...
```

우리 팀이 지정한 스타일링 가이드에도 `--bg-color` 처럼 변수를 생성했다.

이렇게 생성한 변수는 다양한 값을 더 편하게 사용할 수 있게 해준다.

배경색을 정해야 할 때 배경이 들어가는 요소들 모두에 `#fffffa` 해 주었는데 나중에 바꿔야 한다면 하나하나 찾아서 바꿔야 할텐데, 이렇게 정해두면 `style.css`안의 값 하나만 바꿔주면 모두 바뀌게 된다.

사용자 지정 속성을 사용할 때는 아래와 같이 사용한다

```css
body {
    background-color: var(--bg-color);
    color: var(--primary-text);
}
```

미리 지정된 요소를 `var(변수명)`을 이용해 사용할 수 있다.

### var()

[MDN Web Docs - var](https://developer.mozilla.org/ko/docs/Web/CSS/Reference/Values/var)

이 요소도 익숙하진 않은데 var 변수 선언인가? 싶었다.

`var()`는 CSS에서 사용하는 함수로 CSS변수의 값을 다른 속성의 값으로 지정할 때 이용한다.

이 함수는 아래와 같이 사용한다.

`var( <custom-property-name> , <declaration-value>? )`

첫 번째 매개변수 `<custom-property-name>`은 우리가 사용한 사용자 지정 속성의 이름을 나타내는 식별자이다.

두 번째 매개변수인 `<declaration-value>`는  앞의 속성이 유효하지 않을 때 대신 사용할 대체값이다. 첫 번째 속성 이후에 여러개를 ,등으로 구분해 넣을 수 있다.
