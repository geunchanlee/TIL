# TIL_26/02/07 - 백준 2566 최댓값

자바 적응겸 문제풀기

**문제 링크: [최댓값]**

주어진 수 중 최댓값을 찾고 그 값의 위치를 반환하는 문제다.

최댓값이 여러 개일땐 아무 곳이나 반환하면 된다.

입력은 공백으로 구분된 9개의 숫자 리스트가 9줄 주어진다.

처음 문제를 보곤 2차원 배열로 다 담아서 해결할까 했다. 하지만 중요한건 최댓값의 위치와 최댓값을 찾는 것이기 때문에 전역변수 최댓값과 최댓값 인덱스를 만들어두고 한 줄씩 입력받으면서 검사하면 다차원 배열 생성 없이 해결이 가능했다.

문제 해결 방법을 떠올린 뒤 자바 코드로 작성하는게 어려웠다. 아직 자바 코드가 손에 익지 않아서 구현을 어떻게 할까 고민하다가 일단 파이썬 코드로 먼저 알고리즘을 구현한 뒤에 자바 코드로 옮기려면 어떻게 하는지 찾아보면서 해보자는 생각을 했다.

### Python 코드

```python
import sys

maxNum = -1
maxIndex = [0, 0]
for i in range(9):
    numbers = [int(i) for i in sys.stdin.readline().split()]
    if max(numbers) > maxNum:
        maxNum = max(numbers)
        maxIndex[0] = i + 1
        maxIndex[1] = numbers.index(maxNum) + 1

print(maxNum)
print(*maxIndex)
```

---

최댓값을 저장할 변수 `maxNum`과 그 위치를 저장할 리스트 `maxIndex`를 만들었다.

그 다음 한 줄씩 입력받으면서 최댓값보다 큰 수가 입력받은 한 줄 리스트에 존재하는지 검사하고 있다면 최댓값, 위치를 업데이트한다. for문의 i로 2차원 배열 기준 행의 위치와 numbers리스트의 해당 값 인덱스로 열 정보를 알 수 있다. 

이제 이 코드를 자바로 옮겨봤다.

### Java 코드

```java
import java.util.ArrayList;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int maxNum = -1;
        int[] maxIdx = {0, 0};

        for (int i = 0; i < 9; i++) {
            ArrayList<Integer> arr = new ArrayList<Integer>();
            // 한 줄 입력받기
            for (int j = 0; j < 9; j++) {
                arr.add(sc.nextInt());
            }
            // 최댓값 검사
            for (int k : arr) {
                if (k > maxNum) {
                    maxNum = k;
                    maxIdx[0] = i + 1;
                    maxIdx[1] = arr.indexOf(k) + 1;
                }
            }
        }

        System.out.println(maxNum);
        for (int i : maxIdx) {
            System.out.println(i);
        }
    }
}

```

---

우선 가장 기초.

1. 백준에서 자바 코드 제출 시 ***Main.java*** 파일을 입력받는다는 기준으로 제출한다.
클래스 이름을 Main으로 해줬다. 실제로는 패키지 경로도 있지만 제출 시엔 지웠다.
2. Array에서 최댓값을 찾기 위한 `indexOf`는 ArrayList에서 작동하는 메서드이다. Array 대신에 ArrayList를 사용했다.
3. 배열 학습 중 습득한 for-each문도 사용했다. 사실 IntelliJ를 사용하니까 for문을 이렇게 쓸 수 있다고 알림이 떠서 고쳤다.

자바로 코드를 옮기면서 파이썬과 다르게 입력받는 부분부터 어떻게 해야 하는지 고민했다. 배열을 다루면서도 파이썬 코드의 `max`같은 기능을 구현하기 위해 자바에서 `indexOf`로 인덱스로 찾았던 기억은 있는데 `Array`로 코드를 짜니까 작동하지 않아서 찾아보니 `ArrayList`에서 사용 가능한 메서드였다.

가장 기본 중 기본인 입력받기, 배열 다루기, 출력하기 등을 익숙해질 수 있게 해야겠다.

이번 문제에서는 스캐너를 사용해 입력받은 결과도 정답 맞추기에 오류 없이 동작했다. 파이썬에서도 input으로 입력받아도 되는 문제들이 있지만 복잡한 문제의 경우 시간을 줄이기 위해 `sys.stdin.readline()` 을 사용해야 해결 가능했다. 자바 역시 빠른 입력을 위해 Scanner 대신 `BufferedReader`를 사용한다고 한다. 추후 문제를 풀어나가면서 코드가 익숙해지면 사용해봐야겠다.
